# Super-Store-Sales-Dashboard
This repository showcases an end-to-end Power BI data analytics project using the SuperStore dataset. It covers data cleaning in Power Query, KPI creation, interactive dashboards, and sales forecasting. Includes insights on sales, profit, delivery, category trends, top states, and YOY performance.
